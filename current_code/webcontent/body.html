Welcome

Welcome to opendias. The Document Imaging Archive System.

Opendias has been developed to help every household move 'paperless'. Use it to store correspondence that you would normal receive in a paper format - eg Statements, Bills, Letters, Pay Slips, etc...

Key Features

There are a number of ways to help you keep track of your documents once they are archived:

  • OCR (optical character recognition) of the document contents, that can be searched on;
  • Assign documents with multiple 'Tags', making it easier to find and group documents;
  • Search by document date;

Also, we have added some useful features while you're viewing or editing your document:

  • Have the OCR'ed text read to you;
  • Zoom in on a sliding multiple pages view of your document;
  • Live saves back to the database (no 'save button');

Architecture

The openDIAS architecture has been designed to allow many users to access the system.

  • The application connects to 'SANE' which gives you access to various scanning devices, both locally or via the network (see SANE config setting for instructions how).
  • Scanned images and the main database are stored on the server.
  • Clients can connect from any location that can see the server, using a standard web browser.
  • Access controls can be set up by 'clients location' or 'traditional username and password'.

Coming soon...

We are planning on adding more features just as soon as we can. See a selection below. If there is anything that you feel would help, do let us know.

  • Document printing direct from the view page;
  • A 'For Attention' flag that can be used in searches, and shows in document lists;
  • Location based and/or username/password access provisions;
  • Document encryption, to keep your data safe;
